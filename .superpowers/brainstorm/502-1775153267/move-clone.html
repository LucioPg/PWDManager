Flusso:
1. Checkbox nelle righe della tabella + header checkbox (select-all)
2. Barra bulk actions appare sopra la tabella quando ≥1 riga selezionata
3. "Move to..." / "Clone to..." aprono un dialog con Combobox per scegliere il vault di destinazione
4. Move: aggiorna vault_id delle password selezionate
5. Clone: crea nuove password (stesso contenuto) nel vault di destinazione con nuovi nonce
6. "Clone" mostra la lista delle password che verranno clonate nel dialog
